
# Landing Page

A clean and responsive landing page built with **HTML5** and **CSS3**, based on The Odin Project's Landing Page design challenge.

## 📖 Overview

This project demonstrates the fundamentals of modern web development by creating a fully responsive landing page using semantic HTML and Flexbox. It features a hero section, information cards, testimonial section, call-to-action banner, and footer.

## ✨ Features

- Responsive design
- Semantic HTML5 structure
- CSS Flexbox layout
- Modern and clean UI
- Mobile-friendly layout
- Easy to customize

## 🛠 Technologies Used

- HTML5
- CSS3
- Flexbox

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone https://github.com/daplag10/landing-page.git
```

2. Navigate to the project folder.

```bash
cd landing-page
```

3. Open `index.html` in your browser.

No installation or dependencies are required.
